# TUI (Текстовый интерфейс)

Запуск интерфейса:

```bash
anytask-tui
```

Если вы хотите запустить TUI в режиме отладки (чтобы видеть логи операций):

```bash
anytask-tui --debug
```

## Экран входа

При первом запуске (если не настроен автологин) вы увидите форму входа.

- Введите логин и пароль.
- Опция **"Use saved session"**: попытаться использовать прошлую сессию (cookies) без ввода пароля.

Если включен **"Auto-login with session"**, то этого вход будет автоматическим без экрана логина.

## Основной интерфейс

Экран разделен на две части:

1. **Слева**: Список ваших курсов.
2. **Справа**: Рабочая область с вкладками (Tasks, Queue, Gradebook, Export).

### Управление

Нажмите `?` для полной справки, поддерживается навигация стрелками, мышкой и клавишами.

### Вкладки

Переключайтесь между вкладками цифрами `1`, `2`, `3`, `4` или мышкой.

1. **Tasks (Задачи)**:
    - Список всех задач курса.
    - Показывает ваши баллы, дедлайны и статус проверки.
    - Нажмите `Enter` на задаче, чтобы открыть подробное условие.

2. **Queue (Очередь)**:
    - Только для преподавателей.
    - Список решений, требующих проверки.
    - Можно фильтровать по студенту или названию задачи (`Ctrl+F` для фокуса на фильтре).
    - `Enter` открывает полное ревью решения (комментарии, файлы).

3. **Gradebook (Ведомость)**:
    - Таблица с оценками всех студентов.
    - Поддерживает фильтрацию по группам.

4. **Export (Экспорт)**:
    - Настройка экспорта данных.
    - Выберите тип данных (Tasks, Queue, Gradebook).
    - Выберите формат (JSON, CSV, Markdown).
    - Нажмите **Export** для сохранения.

## Горячие клавиши

| Клавиша             | Действие                            |
| ------------------- | ----------------------------------- |
| `Tab` / `Shift+Tab` | Переключение фокуса между панелями. |
| `Ctrl+Q`            | Выход из программы.                 |
| `Ctrl+L`            | Выйти из аккаунта (Logout).         |
| `?`                 | Показать справку по клавишам.       |
| `/`                 | Быстрый переход к строке фильтра.   |
